
Special
========

Name: special
Authors: Andrey Ivnitsky
Drupal: 7.x

Description
===========

The module "version for the special people" to the Russian realities.

Installation
============

1. Place whole "special" folder into your Drupal modules/ or better
   sites/x/modules/ directory.

2. Enable the "special" module by navigating to
     Configuration > Modules
		 
3. Go to admin/appearance/settings and set "special" theme

4. To main theme set block "special version block"

5. To "special" theme set block "normal version block"
